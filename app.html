HTML and CSS Summary

-HTML and CSS are both computer "languages" with syntax and rules.
-HTML controls the "structure" of a web page while CSS controls the "style" of the page.
-HTML can be structured and unstructured.

Stage 2: Adding CSS Style and HTML Structure

What is CSS

-CSS stands for Cascading Style Sheets
-CSS is a sheet that describes various HTML variables and how we want them displayed. It is used to define styles for a variety of purposes for webpages.

Verifying HTML and CSS

-Use these links below to verify your HTML and CSS codes.
-The verification tool will pinpoint areas where your code has errors or need modification.

  • To verify HTML: http://validator.w3.org/#validate_by_input
  • To verify CSS: http://jigsaw.w3.org/css-validator/#validate_by_input
